# PathoPredict Analytics — Algorithmic Inference Platform

PathoPredict Analytics is a decision-making web platform designed to estimate and classify pathological risks with mathematical precision.

The application utilizes an ensembled **Random Forest classifier** trained on **4,920 biological profiles** and **132 clinical symptoms** to predict **41 distinct pathologies**. It features an interactive, mobile-responsive UI with a bilingual search translation engine and targeted, step-by-step recovery recommendations.

---

## Technical Stack

- **Backend:** Django (Python)
- **Machine Learning Moteur:** Scikit-Learn, Joblib, Pandas, NumPy
- **Frontend:** HTML5, CSS3 (Custom Glassmorphism theme), Bootstrap 5, Bootstrap Icons
